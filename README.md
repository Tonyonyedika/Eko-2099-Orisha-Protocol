# Eko 2099: The Orisha Protocol

**An Afrofuturist RPG & Narrative Engine powered by Ancestral AI.**

Set in a futuristic Lagos built on solar-rafts, **Eko 2099** puts you in the role of a **Cyber-Griot**. Your mission is to navigate the Obalende hubs and the Third Mainland Data-Bridge to purge a digital virus (the Esu-Blight) from the city's nervous system—the **Orisha Protocol**.

## 🌟 Hackathon Highlights (AI Leverage)
This project features a **Neuro-Symbolic Narrative Engine** that bridges a deterministic story graph with a dynamic LLM Game Master.
- **Ancestral Wisdom API**: Uses **Groq (Llama 3.1)** to generate context-aware outcomes for custom player actions.
- **State-Aware AI**: The AI doesn't just write text; it modifies your **Life-Force** and **Ase** (spiritual authority) based on your choices.
- **Authentic Atmosphere**: Prompts are tuned for a blend of formal English and Nigerian Pidgin for a true Lagosian feel.

## 🕹️ Controls & Mechanics
- **Story Navigation**: Click to advance through the "Griot Data-Stream."
- **Decision Tree**: Choose from numbered options to navigate the deterministic plot.
- **Custom Actions**: Type anything into the input box to call upon the **Orishas** (AI Mode).
- **HUD Management**:
    - **Life-Force**: Your bio-integrity. Don't let it hit zero!
    - **Ase**: Your spiritual resonance with the city's network.
    - **Neural-Nodes**: Technically advanced disposable code-shards.
    - **Naira**: The currency of Eko.

## 🚀 Setup & Running

1.  **Configure API**: 
    Add your Groq API Key to `.env.local`:
    ```text
    GROQ_API_KEY=your_key_here
    ```

2.  **Install & Run (Windows)**:
    Open PowerShell in the project directory:
    ```powershell
    .\setup.ps1
    .\.venv\Scripts\python.exe main.py
    ```

## 🏗️ Technical Architecture
- **Language**: Python 3.10+
- **Graphics**: Pygame (CE) / Pygame_GUI
- **AI Stack**: Groq SDK / Pydantic (for JSON-schema enforced AI responses)
- **Design**: Modular Theme System (JSON-driven plot and aesthetics)
